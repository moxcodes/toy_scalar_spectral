Detailed Description

Function object for difference of legendre polynomial n and n-2.

A function object representing the function \(q_n(x) \equiv P_n(x) - P_{n-2}(x) \) with its derivative. Used primarily for Newton-Raphson in Gauss-Lobatto generation.
